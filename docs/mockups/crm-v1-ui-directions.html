Option A · Airbnb Calm

Светлый, воздушный, дружелюбный. Хорош для CRM, которой будут пользоваться каждый день без ощущения “корпоративной системы”.

Option B · Studio Desk

Более профессиональный SaaS-вид: много таблиц, фильтров, задач, статусов. Лучше, если CRM быстро станет операционным центром команды.

Option C · Editorial OS

Чуть более авторский и “архитектурный”: хорошо для контент-фабрики и проектных карточек, но требует аккуратности с плотными таблицами.
